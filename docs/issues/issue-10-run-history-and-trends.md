## What to build

A run history view showing all past test runs with pass/fail trends over time. Visual chart (pass/fail rate by day/week). Run comparison view highlighting which tests changed status between two runs (new failures, new passes, still failing). Filtering by project, suite, date range.

## Acceptance criteria

- [ ] Run history page lists all past runs with date, trigger type, verdict
- [ ] Pass/fail trend chart (pass rate over time, adjustable range)
- [ ] Run comparison: side-by-side showing tests that changed status
- [ ] Filter runs by project, suite, date range
- [ ] Tests: history queries, trend data calculations
